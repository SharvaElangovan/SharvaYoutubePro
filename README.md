# SharvaYoutubePro

Automated YouTube quiz video generation and upload system.

## Database Backup

The question bank database (~750k questions, ~194MB) is backed up here:

**Google Drive:** https://drive.google.com/drive/folders/1g-wtQde13DutWT5Xc0x06ryQFJWgBR1L

### Restore Instructions

1. Download `sharva_youtube_pro.db` from the Google Drive link above
2. Place it in: `~/.local/share/com.sharva.youtube-pro/sharva_youtube_pro.db`

```bash
mkdir -p ~/.local/share/com.sharva.youtube-pro/
cp ~/Downloads/sharva_youtube_pro.db ~/.local/share/com.sharva.youtube-pro/
```
